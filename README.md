# Portal de Formularios Bocadeli PWA

Este portal permite acceder a los formularios de KoboToolbox de forma centralizada.

## Instrucciones:
1. Sube todos los archivos a un repositorio de GitHub.
2. Activa **GitHub Pages** en la pestaña Settings.
3. Asegúrate de que el archivo del logo se llame `logo.png`.
4. Para editar formularios, modifica `formularios.json`.
